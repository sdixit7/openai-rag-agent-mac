# OpenAI RAG Agent on macOS

## Overview
A beginner-friendly local AI chatbot built on macOS using OpenAI models, Chroma vector DB, and Streamlit.

## Features
- Chat with OpenAI model
- Ask questions over local PDFs/text files
- Simple tool-routing agent
- Local UI with Streamlit

## Tech Stack
- Python
- OpenAI API
- Streamlit
- Chroma
- PyPDF
- Git / GitHub

## How to Run
1. Clone the repo
2. Create virtual environment
3. Install requirements
4. Add `.env`
5. Run ingestion
6. Start Streamlit

## Sample Questions
- Summarize the PDF in my docs folder
- What does my notes file say about Kubernetes?
- What is 120 * 45?
